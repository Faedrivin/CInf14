\textbf{Lösung}

Wesentliche:
\begin{itemize}
	\item $\comp a \comp c \comp d$
  \item $bcd$
\end{itemize}

Spalten-/Zeilendom.:
\begin{itemize}
	\item $\comp b \comp d$
  \item $acd$
\end{itemize}

Gesamt: $f(a,b,c,d) = \comp a \comp c \comp d + \comp b \comp d + bcd + acd$
